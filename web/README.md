Web (Next.js) - Emotion Insight Dashboard

This is the React UI. It calls the local model backend through Next API routes:
- /api/predict -> backend /predict

Prereqs
- Node 18+

Setup
1) Install deps:
   npm install

2) Configure backend URL (optional):
   export MODEL_API_URL=http://127.0.0.1:8000

3) Run:
   npm run dev

Open:
- http://localhost:3000
